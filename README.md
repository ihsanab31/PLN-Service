# PLN Sevice 
Adalah aplikasi untuk pelanggan melaporkan gangguan yang telah terjadi, untuk memudahkan pelanggan dalam melakukan proses keluhan, yang bertujuan untuk meningkkatkan pelayanan untuk pelanggan.
